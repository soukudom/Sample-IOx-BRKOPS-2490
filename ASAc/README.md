# ASAc Getting Started Example
ASAc enables to run firewall application directly on Catalyst 9000 switch. The aim of this example is to show app installation steps together with required network configuration.

## Installation
Full installation steps are documented [here](https://developer.cisco.com/docs/app-hosting/introduction/)

In general you need to follow these steps:
1. Download ASAc app and configuration packages (day0-config.txt, interface-config.txt)
2. Install the ASAc app
3. Update the IOS-XE and ASA configuration using commands available here (asac-sample-dayN-config.txt, switch-sample-dayN-config.txt). To access ASAc console run `lina_cli` to move from linux CLI to ASAc.
